### Installation

```sh
$ cp .ruby-gemset.sample .ruby-gemset
$ cp .ruby-version.sample .ruby-version
$ cp config/database.yml.sample config/database.yml
```

Use:
```sh
$ rake db:seed
```
to populate development database.
